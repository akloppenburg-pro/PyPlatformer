This file contains information and requirments on how to build the game.
Requirments:
  1. Ensure you are running on a supported OS: Mac, Windows, or Linux.
  2. Python 3 is installed. You can get this here: https://www.python.org/downloads/
  3. Install Pygame. Instructions can be found here: http://kidscancode.org/blog/2015/09/pygame_install/
  4. Run option 1: Load the 'Build' folder in any Python supporting IDE and run the 'Run.py' file. 
  5. Run option 2: Directly run the 'Run.py' through cmd or terminal.
